# credit-risk-assessment
This project predicts creditworthiness and loan default risk using machine learning and neural network models. Multiple algorithms including Logistic Regression, MLP, Random Forest, LSTM, GRU, and Autoencoder are implemented and compared to identify the most accurate model for credit scoring and financial risk prediction.
This project predicts whether a loan applicant is high risk or low risk using machine learning models.

Dataset: bankloans.csv

Models used:
- Logistic Regression
- Random Forest
- MLP
- LSTM
- GRU
- Autoencoder

Tools: Python, Machine Learning, Neural Networks
